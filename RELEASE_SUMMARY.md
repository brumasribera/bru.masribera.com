## 🚀 Quick Deploy Commands

### Vercel (Recommended)
```bash
npm i -g vercel
vercel --prod dist
```

### Netlify
```bash
npm i -g netlify-cli
netlify deploy --prod --dir=dist
```

### GitHub Pages
```bash
npm install -g gh-pages
gh-pages -d dist
```

### Manual Upload
Upload contents of `dist/` folder to any web hosting provider.
